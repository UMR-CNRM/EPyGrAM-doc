epygram.extra : independant side packages, used by epygram¶

Extra side modules: modules needed for some optional purposes of epygram, developped along and distributed with it, but that may be used from outside.

  • falfilfa4py — Interface functions to FA/LFI/LFA formats
  • ialsptrans4py — Interface functions to IAL spectrals transforms
  • pyexttiff — Classes to read Tiff
  • usevortex — Proxies to reach resources using Vortex
  • griberies — utilities around GRIB
