epygram.cli — Command-line interface¶

EPyGrAM Command-Line Interface

Args catalog:¶

  • epygram.cli.args_catalog — Catalog of command-line arguments
    • Functions
    • Catalogs

Sub-commands:¶

  • cartoplot

    An EPyGrAM tool for simple plots of meteorological fields from a resource.

  • convert

    An EPyGrAM tool for converting file formats. Spectral fields are converted into gridpoints.

  • ddhlfa_plot

    An EPyGrAM tool for simple plots of meteorological fields from a DDHLFA.

  • delfield

    An EPyGrAM tool for removing field(s) from a resource.

  • domain_maker

    An interactive EPyGrAM tool for defining a LAM domain, visualize it, and generate the needed namelist blocks.

  • fa_sp2gp

    An EPyGrAM tool for transforming spectral fields from a FA to gridpoint.

  • histogram

    An EPyGrAM tool for making histograms of meteorological fields from a resource.

  • movefield

    An EPyGrAM tool for moving field(s) from a resource to another (fields are not removed from the source file).

  • plot3d

    An EPyGrAM tool for simple 3d plots of meteorological fields from a resource using vtk.

  • point

    An EPyGrAM tool for extracting from a resource the value of meteorological fields on a geographical point.

  • profile

    An EPyGrAM tool for extracting (and plotting) vertical profiles of meteorological fields from a resource.

  • run_epyweb

    A web interface for plotting with epygram and vortex.

  • section

    An EPyGrAM tool for extracting (and plotting) vertical sections of meteorological fields from a resource.

  • spectrum

    An EPyGrAM tool for computing DCT spectrum of meteorological fields (or difference of fields) from a resource.

  • stats

    An EPyGrAM tool for computing basic statistics on meteorological fields.

  • what

    An EPyGrAM tool for asking what’s inside a resource.

  • what_the_grib

    An interface to ask GRIB definitions.
