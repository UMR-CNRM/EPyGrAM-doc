About the package¶

Enhanced Python for Graphics and Analysis of Meteorological fields¶

epygram is a package of classes, designed to handle Meteorological Fields, and various resource Formats from which the Fields can be extracted.

It is distributed along with a series of applicative tools using the package, that can be used as command line tools, or can be taken as templates for building more complex applications with the epygram package.
